# Calculadora-digital
Proyecto 1: Creación de una calculadora inteligente con operaciones básicas y conversión de unidades con historial de operaciones

**Objetivo**
Crear un codigo de calculadora que contenga:
* Menu principal

* Calculos metematicos

* Conversiones

* Sistema de historial

### Caracteristicas

1. Entendible

   A lo largo del codigo se encuentran partes de comentarios que facilita el entendimiento del codigo, junto con un orden en la
    esctructura que permite una buena lectura del mismo

 2. Aprueba de errore

      El codigo cuenta con bucles **try-expect** que ayudan a evitar que el codigo se "cierre" cuando se cometa un error

3. Control
  
      El codigo tambien maneja ciclos **while True** que evitarn que el codigo falle duarente el tiempo del usuario

  ### Manual de uso 

  **Menu principal** 💬

- Calculadora: Menu con las operaciones matematicas a realizar

 * Historial: Listado de todas las operaciones realizadas 

  * Salir: Cierra el programa

  **¿Como hacer una operacion?** ❓

  - Elegir operacion: Al momento de iniciar se abre un meno con las operaciones que puedes realizar !elgie la que quieras¡

  * Primer numero: Despues de elegir operacion, escribe un numero puede ser entero o decimal

  * Segundo numero: Despues de escriber el primer numero haz lo mismo que en el paso anterior

  > !Lo lograste¡ el resultado aparecera despues de esto 💪

**Historial** 📘

> Despues de la operacion puedes seleccionar ver el historial desde el menu principal

- Resultados: Podras ver un listado de Operaccion=Resultado por cada operacion que hallas echo

> Si no has ningun calculo el sistema te avisara
